# tofu-playground
A learning playground for modern IaC

## Whats in here
- `bootstrap/` - one-time setip scripts (state bucket, SOPS keys, etc.)
- `infra/` - OpenTofu modules and Terragrunt live configuration
- `ansible/` - configuration management roles and playbooks
- `docs/modules/` - learning notes for each modules
- `docs/decisions/` - architecture dicision records
